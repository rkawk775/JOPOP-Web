### 🍀 JOPOP-Web
---
<br>
⚙️ 개발 환경 : eclipse, tomcat, github <br>
💻 개발 언어 : oracle, java, javascript, css <br>
📜 프로젝트 설명 : <br> <br>

팝업 스토어를 탐색하고 예약부터 결제까지 한 번에 진행할 수 있는 **웹 기반 예약 플랫폼** 을 개발하였다. <br>
사용자가 다양한 팝업 스토어 정보를 쉽게 확인하고 원하는 장소를 편리하게 예약할 수 있도록 **탐색·예약·결제·후기 관리 기능을 통합**하였다.

사용자는 회원가입 및 로그인 후 메인 페이지에서 팝업 목록 조회, 인기순 정렬, 검색 기능을 활용할 수 있으며, <br>
지도 기반 위치 탐색을 통해 지역별 팝업 정보를 확인할 수 있도록 구현하였다.

관심 팝업(찜) 기능을 통해 즐겨찾기 및 예약/결제가 가능하며, <br>
마이페이지에서는 예약 내역, 결제 내역, 회원 정보 관리, 후기 작성 및 수정 기능을 제공하도록 구성하였다.

팝업 상세 페이지는 상세 정보, 위치 지도, 후기 조회, 찜하기, 예약 및 결제 페이지 이동 기능을 제공하여 사용자가 팝업 탐색부터 방문 예약까지 자연스럽게 이어질 수 있도록 설계하였다.
